B-GLIRT-lavaan

Supplementary Material

This repository contains the supplementary material accompanying the paper:

A Unified Framework for Jointly Modeling Response Times and Item Position Effects in Computer-Based Learning Assessments

### Repository Contents

This repository provides the R/lavaan implementation of the proposed methodology, including:

* B-GLIRT model specification in lavaan;
* single-group models (MOD1, MOD2, MOD3);
* single-group models with covariates;
* multi-group extensions;
* example simulated dataset;
* estimation scripts reproducing the analyses presented in the paper.

### Requirements

The code was developed in **R** using the **lavaan** package.

Main dependency:

* lavaan

### Repository Structure

```text
├── Supplementary Material_B_GLIRT.R
├── dataset_SM.RData
└── README.md
```

### Reproducibility

The scripts reproduce the estimation of all models presented in the supplementary material. The simulated dataset is provided solely for illustrative purposes and allows users to replicate every analysis reported in the supplementary document.
